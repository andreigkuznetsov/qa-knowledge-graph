# AI Workspace

This directory contains reusable instructions, standards, and task specifications for AI-assisted development of the QA Knowledge Graph project.

## Directory Structure

### `MASTER_PROMPT.md`

The permanent engineering guidelines for AI agents working on the project. This document defines architectural principles, coding standards, development workflow, and quality expectations.

### `phases/`

Task-specific engineering specifications. Each phase describes a discrete implementation objective and should be executed together with `MASTER_PROMPT.md`.

### `reviews/`

Review specifications and quality checklists for architecture, code quality, testing, performance, and other engineering aspects.

## Development Workflow

1. Read `MASTER_PROMPT.md`.
2. Select the appropriate document from `phases/`.
3. Analyze the current implementation.
4. Implement the required changes.
5. Run all relevant tests.
6. Summarize the completed work, including architectural decisions and remaining technical debt.
